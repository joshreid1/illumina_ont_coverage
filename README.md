# illumina_ont_coverage

Code and analysis scripts associated with the paper: **"Lessons for epilepsy clinical genomics: sequencing dark 
regions in patients with developmental and epileptic 
encephalopathies"**

## Overview

This repository contains the scripts and workflows used to analyse and compare
sequencing coverage between Illumina short-read and Oxford Nanopore Technologies
(ONT) long-read sequencing data, as described in the accompanying publication.
